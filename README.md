# 3D Realms Games Pack for DREAMM
## How to use this .dreammpack with DREAMM
1. Download the latest here: https://github.com/M-A-Boucher/sierra-dreamm/releases/latest/download/3drealms.dreammpack
2. In DREAMM 4.0x12 (or later), select **Manage DREAMMPacks...** from the cog menu. 
3. Drag and drop the 3drealms.dreammpack you just downloaded onto the window.

Alternatively:

3. Click on **View DREAMMPack Folder**.
4. Copy the 3drealms.dreammpack file to this folder, close and relaunch DREAMM.
